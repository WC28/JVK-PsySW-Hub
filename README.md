# JVK-PsySW-Hub

PSW Hub Pitch is a modern one-page web prototype for a Psychiatric Social Work Management System.
It is designed to present the concept of a digital platform that helps social workers manage cases, track workload, monitor progress, and improve team coordination in a more efficient way.

## Project Purpose

This project is a prototype landing page used to present the vision of **PSW Hub** — a digital solution for psychiatric social work services in Thailand.

The goal is to make social work operations:

* easier to manage
* more organized
* more visual
* more accessible on both desktop and mobile

## Included in This Prototype

The prototype presents the idea of a full system with modules such as:

* Login system with 3 roles: Admin / Social Worker / Public
* Dashboard with statistics cards and charts
* Intake form for new case submission
* Case management table with status tracking
* Case conference section
* Community preparation progress tracker
* KPI dashboard
* Social worker workload overview

Note: This repository currently contains the **pitch/prototype page**, not the full backend system.

## How to Run Locally

You can open the project locally in several ways:

1. Open the HTML file directly in your browser, or
2. Use VS Code with Live Server

If using Live Server, the local URL may look like:

`http://127.0.0.1:5500/psw-hub-pitch.html`

Important: `127.0.0.1` only works on your own computer. Other devices cannot access that link unless the project is deployed online or shared through your local network IP.

## Deploy with GitHub Pages

To make the site accessible on phone or for other people:

1. Upload the project files to a GitHub repository
2. Rename the main file to `index.html` if needed
3. Go to **Settings > Pages**
4. Set:

   * Source: `Deploy from a branch`
   * Branch: `main`
   * Folder: `/(root)`
5. Save and wait for GitHub Pages to publish the site

Your public link will look like:

`https://your-username.github.io/repository-name/`
